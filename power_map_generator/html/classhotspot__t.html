Detailed Description

Class for storing the parameters for a gaussian shaped hotspot to be drawn onto a power map.

Each hotspot is a gaussian distribution with a life cycle in which it spawns onto the power map at a random location with zero initial amplitude, grows by aging rate until the hotspot's amplitude reaches or exceeds its peak amplitude, then shrinks at the same rate until it dissapears and is replaced by a new hotspot.

Member Function Documentation

hotspot_stage_t hotspot_t::advance_age ( )

Move the hotspot one step forward along its normal lifecycle.

Returns
The new value of stage at the end of the function call.

Detailed function behavior:

  1. If this hotspot was GROWING before this function call, then aging_rate is added to amplitude. If the new amplitude >= peak_amplitude, then stage is overwritten with SHRINKING.
  2. If this hotspot was SHRINKING before this function call, then aging_rate is subtracted from amplitude. Of the new amplitude <= 0.0f, then stage is overwritten with DEAD and amplitude is clamped to 0.0f.
  3. If this hotspot was DEAD before this function call, then this function returns immediatly without performing any operation.
