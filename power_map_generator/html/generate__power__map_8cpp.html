Generate a random power map by superimposing a bunch of hills of random sizes at random (x, y) locations which decay at a random rate.

Parameters
[in,out]power_map_stateUsed to track where each hotspot on the power map is. The power map state is then advanced slightly and written back in-place.
[out]power_map2D array that is overwritten with a new heat map.
