enum arg_status_t : unsigned char
strong

Used to indicate weather an parameter was passed via the command line or not.

Enumerator
NOT_FOUND 

The relevant command was neither passed on the command line nor does it have a default value. The corresponding variable in args_t is uninitialized.

FOUND 

The relevant command line parameter was explicitly passed on the command line and successfully parsed.

DEFAULTED 

The relevant command line parameter was not passed on the command line so the corresponding variable in args_t was initialized to a default value.

INVALID 

The user attempted to explicitly pass the relevant command line parameter but the parameter was invalid and could not be parsed.
